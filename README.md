# 🏨 Radisson Red Hotel Analytics Dashboard

An interactive Power BI dashboard project designed to analyze hotel bookings, listing, reviews and overall business performance for Radisson Red.

---

## 📌 Project Overview

This Power BI project focuses on transforming raw hotel data into meaningful insights using interactive visualizations and business intelligence techniques.

The dashboard helps businesses:

- Monitor hotel listing 
- Analyze booking and trends
- Understand customer reviews 
- Track important KPIs
- Improve decision-making using data

---

## 🚀 Features

✅ Booking Analysis  
✅ Hotel Performance Tracking  
✅ Customer Insights  
✅ Occupancy Rate Monitoring  
✅ KPI Dashboard  
✅ Clean & Professional Dashboard Design
